# Drama Frontend

ReelShort-style vertical video drama streaming app - Next.js frontend for AI-generated episodic content.

## Tech Stack
- Next.js 15 with App Router
- TypeScript
- Tailwind CSS
- Zustand for state management
- Framer Motion for animations

## Features
- Vertical scrolling video feed
- Token-based authentication
- Auto-play on scroll
- Smooth transitions
- Mobile-first responsive design

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Environment Variables

Copy `.env.local.example` to `.env.local` and update with your backend API URL:

```
NEXT_PUBLIC_API_URL=your_backend_url
```
